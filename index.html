Privacy Policy for Codeforces Ratings Chrome Extension

Effective Date: 10/07/2024

Introduction

Thank you for using the Codeforces Ratings Chrome extension. This Extension is designed to provide users with the ability to monitor their and their friends' ratings on Codeforces, a competitive programming platform. This Privacy Policy outlines how we collect, use, store, and protect the information you provide while using the Extension.

Data Collection and Usage

User Data: The Extension collects Codeforces user handles that you input to monitor ratings. This data is encrypted locally using AES-GCM encryption before storage.

API Usage: The Extension interacts with the Codeforces API to fetch rating and user information based on the user handles provided. No user data is transmitted to our servers; all API interactions occur directly between your browser and Codeforces.

Storage and Encryption

Local Storage: Encrypted user data (including user handles and associated rating data) is stored locally using Chrome's local storage API. This data is encrypted using AES-GCM encryption, which ensures secure storage within your browser.

Encryption Technology: AES-GCM (Advanced Encryption Standard - Galois/Counter Mode) is utilized for encrypting and decrypting user data. This encryption standard is widely recognized for its security and integrity in protecting sensitive information.

Permissions

Storage Permission: This extension requires permission to securely store your Codeforces user data and preferences using Chrome's storage API (chrome.storage.local). This ensures that your information remains protected and accessible only to you within your browser.

Data Security

Security Measures: We employ industry-standard security measures to protect your data against unauthorized access, alteration, disclosure, or destruction. This includes encryption of sensitive data, regular security assessments, and access controls.

Data Handling

Purpose of Data: User handles are collected solely for the purpose of retrieving and displaying Codeforces rating information within the Extension.

Data Retention: Encrypted user data is retained locally on your device until you choose to clear it through the Extension's interface.

Third-Party Services: The Extension utilizes the Codeforces API to retrieve user information. Please refer to Codeforces privacy policy for details on how they handle user data.

User Rights

Access and Deletion: You have the right to access, correct, or delete your personal information stored within the Extension. This can be managed directly through the Extension's interface.

Changes to this Privacy Policy

Policy Changes: We may update this Privacy Policy from time to time to reflect changes in our practices or for other operational, legal, or regulatory reasons. We will notify users of any material changes through the Extension or via other appropriate means.
